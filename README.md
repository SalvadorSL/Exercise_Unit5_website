# Exercise_Unit5_website
Descripción del repositorio
    Este repositorio contiene los archivos de un sitio web básico desarrollado como parte del ejercicio de la Unidad 5. Incluye la estructura principal de la página, estilos visuales y contenido informativo.

Tecnologías utilizadas
    - HTML: para la estructura del contenido.
    - CSS: para el diseño y la presentación visual.

Cómo abrir el sitio web
    - Descarga o clona el repositorio en tu equipo.
    - Abre la carpeta del proyecto.
    - Haz doble clic en el archivo index.html o ábrelo con tu     navegador preferido (Chrome, Firefox, Edge, etc.).
    - El sitio se mostrará localmente en tu navegador.
